Macのフォルダ内に住むデジタル付箋紙。

The Stykは、デジタルノートをFinderのフォルダに固定（アンカー）する極小のプログラムです。作成したフォルダを開いている間だけノートが画面に表示され、フォルダから移動すると消え、戻ると再表示されます。

## インストール

https://setor101.com.br/apps/styk からThe Stykをダウンロードし、アプリケーション（Applications）フォルダにドラッグして、アイコンをダブルクリックして起動します。

## 使い方

The Stykは、メニューバーの右側にノートのアイコンを表示します。アイコンをクリックしてメニューを表示し、**「New note in this folder（このフォルダに新規ノート）」**を選択して付箋を作成します。文字を入力すると、ノートは自動的に保存されます。

### メニューバー
ステータスバーのメニューには、フォルダごとにグループ化されたすべてのノートが一覧表示されます。任意のノートをクリックすると、Finderでそのフォルダに直接ジャンプしたり、エクスポートや削除を行ったりできます。

### ノートの操作
ノートの上にマウスカーソルを置くと、アクションバーが表示されます。ここでは以下の操作が行えます：
- 付箋の色の変更
- フォントサイズ（A− / A+）およびフォントスタイル（Aa）の調整
- ノートの共有（AirDrop、メッセージ、メールなど）
- ノートの削除

ノートの背景をドラッグして移動させたり、枠線をドラッグしてサイズを変更したりできます。ノートの内部では、`⌘ +` や `⌘ −` を使用してテキストサイズを素早く調整できます。

### 設定（Preferences）
バーメニューから設定（Preferences）を開き、以下を構成できます：
- **Language（言語）**: ポルトガル語（ブラジル）、英語、中国語、日本語、ドイツ語、フランス語から選択できます。
- **Finder Permission（Finderのアクセス許可）**: アクティブなFinderウィンドウを追跡するために必要なApple Events自動化のアクセス許可を管理します。
- **Start at Login（ログイン時に起動）**: Macの起動時にThe Stykを自動的に開くかどうかを切り替えます。
- **Backups（バックアップ）**: 自動的な毎日のローカルバックアップを設定するか、手動ですべてのノートをエクスポート/復元します。

## FAQ（よくある質問）

### 特別なアクセス許可は必要ですか？
はい。最初の起動時に、macOSはFinderを制御するためのアクセス許可を求めます。これは、The Stykがどのアクティブなフォルダが開いているかを検出し、対応するノートを表示するために必要です。誤って拒否した場合は、設定（Preferences） -> 「Request Finder permission...（Finderのアクセス許可を要求...）」ボタンから再度プロンプトを表示させることができます。

### ノートを削除するとどうなりますか？
削除は完全に取り消し可能です。削除されたノートはアプリの内部ゴミ箱（メニューバーからアクセス可能）に入り、5日後に自動的に完全に消去されます。

### フォルダを移動、名前変更、または削除した場合はどうなりますか？
- **移動または名前変更されたフォルダ**: The StykはmacOSのブックマーク機能を使用しているため、フォルダの名前を変更したり別のドライブに移動したりした場合でも、ノートは自動的にそのフォルダに追従します。
- **削除されたフォルダ**: ノートは失われません。メニューの「Orphan notes（孤立したノート）」セクションに移動され、そこで再固定、エクスポート、またはゴミ箱への移動を選択できます。

### macOS 10.xで動作しますか？
メインのApple Silicon版はmacOS 11（Big Sur）以降が必要です。ただし、macOS 10.15（Catalina）以降で動作するIntelレガシー版も用意されています。

### The Stykは標準のスティッキーズとどう違うのですか？
ノートがデスクトップを無期限に乱杂にする標準的な付箋アプリとは異なり、The Stykはノートを特定のフォルダに文脈的に固定します。Finderで実際にそのフォルダを開いて表示しているときにのみ表示されます。
