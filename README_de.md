# The Styk
<img src="assets/logo.png" width="128" alt="The Styk Logo" />

Digitale Notizen, die in Ihren Mac-Ordnern leben.

The Styk ist ein winziges Programm, das digitale Notizen an Ihren Finder-Ordnern verankert. Eine Notiz schwebt auf dem Bildschirm, während Sie sich in dem Ordner befinden, in dem Sie sie erstellt haben – verlassen Sie den Ordner, verschwindet sie; kehren Sie zurück, erscheint sie wieder.

## Installation

Laden Sie The Styk unter https://setor101.com.br/apps/styk herunter, ziehen Sie es in Ihren Programme-Ordner und doppelklicken Sie auf das Symbol, um es zu starten.

## Verwendung

The Styk platziert ein Notizsymbol auf der rechten Seite Ihrer Menüleiste. Klicken Sie auf das Symbol, um das Menü anzuzeigen. Von hier aus können Sie **„Neue Notiz in diesem Ordner“** wählen, um eine Notiz zu erstellen. Schreiben Sie hinein; die Notiz speichert sich automatisch.

### Menüleiste
Das Menü der Statusleiste listet alle Notizen auf, gruppiert nach Ordnern. Klicken Sie auf eine Notiz, um direkt zu diesem Ordner im Finder zu springen, sie zu exportieren oder zu löschen.

### Interaktion mit den Notizen
Bewegen Sie den Mauszeiger über eine Notiz, um deren Aktionsleiste anzuzeigen. Von hier aus können Sie:
- Notizfarben ändern.
- Schriftgröße (A− / A+) und Schriftstil (Aa) anpassen.
- Die Notiz teilen (über AirDrop, Nachrichten, Mail usw.).
- Die Notiz löschen.

Ziehen Sie die Notiz am Hintergrund, um sie zu verschieben, oder an den Rändern, um die Größe zu ändern. Verwenden Sie innerhalb der Notiz `⌘ +` und `⌘ −`, um die Textgröße schnell anzupassen.

### Einstellungen
Öffnen Sie im Leistenmenü die Einstellungen, um Folgendes zu konfigurieren:
- **Sprache**: Wechseln Sie zwischen Portugiesisch (Brasilien), Englisch, Chinesisch, Japanisch, Deutsch oder Französisch.
- **Finder-Berechtigung**: Verwalten Sie die Berechtigungen für die Apple Events-Automatisierung, die zum Rastreieren des aktiven Finder-Fensters erforderlich sind.
- **Beim Anmelden starten**: Legen Sie fest, ob The Styk beim Starten Ihres Macs automatisch geöffnet werden soll.
- **Backups**: Konfigurieren Sie automatische tägliche lokale Backups oder exportieren/wiederherstellen Sie alle Notizen manuell.

## FAQ

### Erfordert dies spezielle Berechtigungen?
Ja. Beim ersten Start fragt macOS nach der Berechtigung zur Steuerung des Finders. Dies ist erforderlich, damit The Styk erkennen kann, welcher Ordner aktiv ist, und die entsprechenden Notizen anzeigen kann. Wenn Sie dies versehentlich ablehnen, können Sie die Eingabeaufforderung über Einstellungen -> Schaltfläche „Finder-Berechtigung anfordern...“ erneut auslösen.

### Was passiert, wenn ich eine Notiz lösche?
Das Löschen ist vollständig umkehrbar. Gelöschte Notizen landen im internen Papierkorb der App (erreichbar über die Menüleiste) und werden nach 5 Tagen automatisch gelöscht.

### Was passiert, wenn ich einen Ordner verschiebe, umbenenne oder lösche?
- **Verschobene/umbenannte Ordner**: The Styk verwendet macOS-Lesezeichen (Bookmarks), sodass Notizen dem Ordner automatisch folgen, selbst wenn Sie ihn umbenennen oder auf ein anderes Laufwerk verschieben.
- **Gelöschte Ordner**: Notizen gehen nicht verloren; sie werden in den Bereich „Verwaiste Notizen“ im Menü verschoben, wo Sie sie neu verankern, exportieren oder löschen können.

### Funktioniert dies mit macOS 10.x?
Die primäre Apple Silicon Version erfordert macOS 11 (Big Sur) oder neuer. Es ist jedoch eine Intel-Legacy-Version verfügbar, die unter macOS 10.15 (Catalina) und neuer läuft.

### Wie unterscheidet sich The Styk von Standard-Haftnotizen?
Im Gegensatz zu Standard-Haftnotiz-Apps, bei denen Notizen Ihren Schreibtisch unbegrenzt überladen, verankert The Styk Notizen kontextbezogen an bestimmten Ordnern. Sie erscheinen nur, wenn Sie diesen Ordner tatsächlich im Finder öffnen und anzeigen.
